# 안드로이드폰에 앱 설치하기 (상세 가이드)

방금 안내해 드린 방법이 정석입니다! 
아래 체크리스트를 하나씩 확인하면서 진행해 보세요.

## 1단계: 컴퓨터 설정
컴퓨터에 **Android Studio**와 **Flutter**가 설치되어 있어야 합니다.
(터미널에서 `flutter doctor`를 쳤을 때 `[✓] Android toolchain`이 체크되어 있으면 OK)

## 2단계: 핸드폰('개발자 옵션') 켜기
핸드폰을 컴퓨터가 인식하려면 **USB 디버깅**을 켜야 합니다.

1. **설정 > 휴대전화 정보 > 소프트웨어 정보** 로 이동합니다.
2. **빌드 번호(Build Number)**를 7번 연속으로 타탁! 터치합니다. ("개발자 모드가 켜졌습니다" 메시지 확인)
3. 뒤로 가기 후 **설정 > 개발자 옵션** 메뉴로 들어갑니다.
4. 스크롤을 내려 **USB 디버깅** 스위치를 **ON**으로 켭니다.

## 3단계: 연결 및 설치
1. 핸드폰을 USB 케이블로 컴퓨터에 연결합니다.
2. 핸드폰 화면에 "이 컴퓨터를 신뢰하시겠습니까? (Allow USB debugging?)" 팝업이 뜨면 **"허용(Allow)"**을 누릅니다.
3. 컴퓨터 터미널(프로젝트 폴더)에서 아래 명령어를 실행합니다.

```bash
flutter run
```

---
## 💡 꿀팁 (매번 케이블 연결하기 귀찮다면?)
`flutter run`은 개발용 임시 실행입니다. 
케이블을 뽑아도 앱이 계속 실행되게 하려면, **설치 파일(APK)**을 만들어서 넣으면 됩니다.

1. **APK 파일 생성**:
   ```bash
   flutter build apk --release
   ```
2. **파일 위치**:
   `build/app/outputs/flutter-apk/app-release.apk` 경로에 파일이 생깁니다.
3. **설치**:
   이 파일을 카카오톡이나 구글 드라이브로 핸드폰에 보내서 설치하면, 앱스토어에서 받은 것처럼 영구적으로 설치됩니다.
