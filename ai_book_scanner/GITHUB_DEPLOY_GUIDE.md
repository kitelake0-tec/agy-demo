# 깃허브로 앱 배포 및 다운로드 방법 (무선 설치)

케이블 연결 없이, 깃허브 서버에서 앱을 만들고(Build) 핸드폰으로 다운로드 받는 방법입니다.

## 1. 설정 방법 (최초 1회)
제가 만들어드린 `.github/workflows/android_build.yml` 파일이 포함된 코드를 깃허브에 푸시합니다.

```bash
git add .
git commit -m "Add GitHub Actions for Android Build"
git push origin master
```

## 2. 다운로드 방법 (매번 배포 시)
코드를 푸시할 때마다 깃허브가 자동으로 앱을 만듭니다.

1. 깃허브 레포지토리 페이지로 이동합니다.
2. 상단 메뉴의 **Actions** 탭을 클릭합니다.
3. 리스트에서 가장 위에 있는 **Build Android App** (또는 커밋 메시지)을 클릭합니다.
4. 화면 아래쪽 **Artifacts** 섹션을 찾습니다.
5. **`ai-book-scanner-apk`**를 클릭하면 `app-release.apk` 파일이 다운로드됩니다.
   - *참고: 다운로드 파일은 zip으로 압축되어 있을 수 있습니다. 압축을 풀면 apk 파일이 나옵니다.*

## 3. 핸드폰 설치
1. 다운로드 받은 APK 파일을 카카오톡, 구글 드라이브 등으로 핸드폰에 보냅니다.
2. 핸드폰에서 파일을 열고 **설치** 버튼을 누릅니다.
   - *"출처를 알 수 없는 앱 설치"* 권한 허용이 필요할 수 있습니다.
