# Interview Scratchpad / 인터뷰 메모 - cx-ai-career-transition

## Context / 맥락
- Date / 날짜: 2026-03-09
- Reference / 참고자료: None
- Topic / 주제: CX 직무에 AI 역량을 더한 새로운 구직 및 포트폴리오 전략

## Emerging Themes / 주요 주제
- CX(Customer Experience) 실무와 AI 기술의 결합
- AI를 활용한 구직 차별화 전략 및 포트폴리오 구성 요소 탐색
- 반복적인 고객 응대(게시판, 채팅)의 자동화 니즈
- 비정형 고객 데이터의 추출 및 구조화 니즈
- **비기술적(Non-technical), 실무 중심의 AI 활용** 포트폴리오 기획
- 외부 연동 툴(Zapier/Make) 대신 LLM 자체의 기능을 극대화하는 심플한 워크플로우 선호
- 포트폴리오용 가상 데이터(Synthetic Data) 기획 및 생성 필요성 대두
- **교육 및 커머스 도메인 타겟팅 포트폴리오**

## Key Insights / 핵심 인사이트
- 사용자는 현재 CX 업무를 수행하고 있음
- AI를 추가하여 구직 시장에서의 경쟁력을 높이고 싶어 함
- **Pain Point**: 반복적인 게시판 및 채팅 응대 업무에 리소스가 많이 들어감
- **Pain Point**: 고객 데이터를 수동으로 추출하고 데이터화하는 과정의 비효율성 존재
- **Constraint**: 사용자는 파이썬 등 프로그래밍을 다뤄본 적이 없으며, 고도의 '기술적(Technical) 포트폴리오'를 구상하는 것은 아님. 노코드 도구나 프롬프트 엔지니어링, 워크플로우 기획 위주의 접근이 필요.
- **Goal**: 기존 CX 매니저 포지션으로 지원하되, AI 도구를 업무 효율화에 적극 활용한 '경험'을 핵심 역량으로 어필하고자 함.
- **Current Stack**: 대고객 채널로 '카카오톡', '사내 어드민 게시판' 사용 중. 데이터 관리는 '구글 스프레드시트'를 수기로 활용해 기록 중.
- **Preference**: Zapier/Make 같은 외부 연동 자동화 툴 학습보다는, LLM(ChatGPT, Claude 등) 자체 기능을 활용하여 실무 문제를 해결하는 직관적인 방식을 선호함.
- **Issue**: 실제 회사 데이터를 반출하여 포트폴리오로 사용할 수 없는 보안/기밀 문제가 있음
- **Target Domain**: 구직을 희망하는 타겟 산업군은 '교육 회사' 또는 '이커머스 회사'임.

## Constraints & Boundaries / 제약사항 및 경계
- 고도의 코딩, 프로그래밍 기반 작업은 배제 (파이썬 등 사용 안 함)
- 복잡한 자동화 툴(Zapier/Make) 의존도를 낮추고, 누구나 따라할 수 있는 LLM 프롬프트/파일 분석 기능 위주의 사용
- 현 직장의 실제 고객 데이터를 포트폴리오에 절대 사용하지 않음 (보안 준수)

## Decisions Made / 결정사항
- 주제를 'CX + AI 커리어 전환 및 포트폴리오 준비'로 확정
- 지원 방향을 "AI 역량을 갖춘 실무형 CX 매니저"로 설정
- 코딩형 포트폴리오 지양, **'문제 해결(Problem Solving)과 툴 활용(Tool Utilization)'** 중심의 포트폴리오 기획으로 방향 선회
- LLM 본연의 기능(파일 업로드, 데이터 분석, 프롬프트 템플릿)을 활용한 데이터 가공 및 응대 템플릿 제작을 핵심 포트폴리오로 설정
- 포트폴리오에 사용할 가상의 고객 리뷰/문의 데이터(Mock Data)를 직접 기획하고 생성하기로 함

## Tensions / Trade-offs / 긴장관계 및 트레이드오프
- 100% 실시간 완전 자동화(API/Zapier) vs 도입 및 관리가 쉬운 수동+AI 결합형 프로세스(LLM 직접 활용) 사이의 선택
- 실제 상황의 복잡성 vs 가상 데이터가 주는 인위적 느낌(얼마나 현실적으로 가상 데이터를 만들 것인가)

- "내 직무는 지금 CX업무를 하고 있어"
- "잠깐 나는 파이썬을 다뤄 본 적이 아예 없어. 기술적인 포트폴리오를 찾는 것은 아니야."
- "Zapier(재피어)나 Make(메이크) 가 꼭 필요한거야? 요즘 LLM을 활용해서도 다룰 수 있지 않나?"
- "근데 내가 지금 가지고 있는 회사 데이터로는 포트폴리오를 만들 수 없잖아... 가상의 상황을 만들어야 될 수 있는데 그런 것들은 어떻게 준비해야되지"
- "교육 회사나 이커머스 회사 쪽으로 생각하고 있어"
