# 바람의 나라 (Web Remake) - 테스트 가이드

이 문서는 `baramstory` 게임을 실행하고 테스트하는 방법을 설명합니다.

## 1. 게임 실행 방법

터미널에서 다음 명령어를 실행하여 개발 서버를 시작하세요:

```bash
npm run dev
```

터미널에 표시되는 로컬 주소(예: `http://localhost:5173` 또는 `http://localhost:5174`)를 브라우저에서 엽니다.

## 2. 게임 플레이 흐름 및 테스트 체크리스트

### A. 인트로 화면
- [ ] 게임 접속 시 "바람의 나라" 타이틀과 인트로 텍스트가 표시되는지 확인
- [ ] "다음" 버튼을 클릭하여 스토리가 진행되는지 확인
- [ ] 마지막에 "모험 시작" 버튼을 클릭하여 직업 선택 화면으로 넘어가는지 확인

### B. 직업 선택 (Character Selection)
- [ ] 4가지 직업(전사, 도적, 주술사, 도사) 카드가 표시되는지 확인
- [ ] 카드에 마우스를 올렸을 때 하이라이트 효과가 있는지 확인
- [ ] 카드를 클릭하면 마을(TownScene)로 이동하는지 확인

### C. 마을 (TownScene)
- [ ] **이동**: 방향키(`↑`, `↓`, `←`, `→`)를 눌러 캐릭터가 부드럽게 움직이는지 확인
- [ ] **그래픽**: 현재는 캐릭터와 타일이 단순한 도형(네모, 동그라미)으로 표시됩니다. (정상 동작)
- [ ] **UI 확인**:
    - 좌측 상단: HP/MP 바, 레벨, 경험치 바가 표시됨
    - 하단: 스킬 단축키바(1~0)가 표시됨
- [ ] **상호작용**:
    - **I 키**: 인벤토리 창 열기/닫기
    - **Q 키**: 퀘스트 창 열기/닫기
    - **K 키**: 스킬 창 열기/닫기
    - **M 키**: 사냥터 이동 창 열기/닫기
    - **ESC 키**: 열린 창 닫기

### D. 사냥터 및 전투 (Dungeon)
- [ ] 마을 상단의 보라색 포탈(원형)에 닿으면 사냥터 이동 여부를 묻거나 이동되는지 확인
- [ ] 몬스터(빨간 네모)가 필드에 돌아다니는지 확인
- [ ] 몬스터 근처에서 `A`키 또는 스킬 키를 눌러 공격이 가능한지 확인

## 3. 현재 알려진 차이점 (vs 원본)
- **그래픽**: 이미지 에셋 대신 기본 도형(Primitive Shapes)을 사용 중입니다.
- **엔진**: Canvas API 직접 구현 방식에서 React + Phaser 엔진으로 변경되었습니다.

## 4. 문제 발생 시
브라우저 개발자 도구(`F12`)를 열어 `Console` 탭에 빨간색 에러 메시지가 있는지 확인해주세요.
