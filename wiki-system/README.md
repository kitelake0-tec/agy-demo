# 사내 위키 시스템 (Internal Wiki System)

Google Apps Script(GAS)와 Google Sheets를 기반으로 한 경량 사내 위키 시스템입니다.

## 주요 기능
- **Markdown 에디터**: Toast UI Editor를 사용하여 편리하게 문서를 작성할 수 있습니다.
- **AND 검색**: 여러 검색어를 띄어쓰기로 입력하여 정교한 검색이 가능합니다.
- **카테고리 분류**: 문서를 카테고리별로 분류하여 관리할 수 있습니다.
- **히스토리 관리**: 문서 수정 시 이전 버전이 History 시트 에 저장됩니다.
- **안전한 데이터 처리**: 날짜 객체 직렬화 에러를 방지하는 방어 로직이 포함되어 있습니다.

## 설치 방법

1.  **Google Spreadsheet 생성**:
    -   새 구글 스프레드시트를 만듭니다.
    -   하단 탭에서 시트 이름을 각각 `Docs`와 `History`로 변경합니다.
    -   `Docs` 시트의 첫 번째 행(헤더)에 다음 항목을 입력합니다:
        `ID`, `Title`, `Content`, `Category`, `Format`, `Author`, `Updated_at`, `Status`
    -   `History` 시트의 첫 번째 행(헤더)에 다음 항목을 입력합니다:
        `History_ID`, `Doc_ID`, `Old_Content`, `Modifier`, `Modified_at`

2.  **Apps Script 프로젝트 생성**:
    -   스프레드시트 도구 모음에서 `확장 프로그램` > `Apps Script`를 클릭합니다.

3.  **코드 복사**:
    -   기존의 `Code.gs` 내용을 이 저장소의 `Code.gs` 내용으로 교체합니다.
    -   `파일` > `+` > `HTML`을 선택하여 이름을 `index`로 만들고, 이 저장소의 `index.html` 내용을 붙여넣습니다.

4.  **배포**:
    -   우측 상단의 `배포` > `새 배포`를 클릭합니다.
    -   유형을 `웹 앱`으로 선택합니다.
    -   설명: `사내 위키 시스템 v3.1`
    -   다음 사용자로 실행: `나`
    -   액세스할 수 있는 사용자: `조직 내부` (또는 필요에 따라 설정)
    -   `배포` 버튼을 누르고 권한 승인을 완료합니다.

5.  **사용**:
    -   생성된 웹 앱 URL로 접속하여 위키를 사용합니다.
