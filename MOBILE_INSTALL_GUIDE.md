# 앱 다운로드 및 설치 가이드 (모바일 실행)

현재 GitHub Actions를 통해 자동으로 빌드된 APK 파일을 다운로드하여 설치할 수 있습니다.

## 설치 방법

1. 이 저장소의 **Actions** 탭으로 이동합니다.
2. 최신 **Build Android App** 워크플로우 실행 기록을 클릭합니다.
   - *참고: 초록색 체크 표시(✅)가 있는 가장 최근 항목을 선택하세요.*
3. 화면 하단의 **Artifacts** 섹션에서 `ai-book-scanner-apk`를 클릭하여 다운로드합니다.
4. 다운로드된 zip 파일의 압축을 풀면 `app-release.apk` 파일이 있습니다.
5. 이 파일을 안드로이드 휴대폰으로 전송한 후 실행하여 설치합니다.

## 주의 사항
- 설치 시 휴대폰에서 **"출처를 알 수 없는 앱 설치"** 권한 허용이 필요할 수 있습니다.
- Google Play Protect 경고가 뜰 경우 "무시하고 설치"를 선택해야 할 수 있습니다 (서명되지 않은 앱인 경우).
